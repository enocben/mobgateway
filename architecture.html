Mobile Money Gateway — Architecture

Plateforme unifiée de paiement mobile money · AdonisJS 7 · Inertia.js · React · PostgreSQL

Tech Stack

  • • AdonisJS 7 — Framework HTTP
  • • Inertia.js — SSR sans API
  • • React 19 + shadcn/ui
  • • PostgreSQL + Lucid ORM
  • • VineJS — Validation

Key Features

  • • Routage MSISDN → Provider
  • • Providers extensibles code-first
  • • Auto-discovery + sync DB
  • • Multi-application multi-tenant
  • • Webhooks + audit logs

Data Model

  • • App → Countries → Operators
  • • Operators ↔ Providers (routes)
  • • Countries ↔ Providers (pivot)
  • • Transactions + Events
  • • Commissions + Ledger
